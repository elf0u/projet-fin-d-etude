# IoT AgriLink

IoT AgriLink is a smart agriculture project that combines IoT, artificial intelligence, and a mobile application to help users monitor agricultural conditions and make better decisions.

## Features

- IoT sensor data collection
- Temperature, air humidity, and soil humidity monitoring
- MQTT communication
- MySQL data storage
- PHP backend APIs
- Flutter mobile application
- Historical charts and alerts
- AI-based temperature prediction
- Plant recommendation system
- Plant disease diagnosis using TensorFlow Lite
- AgriLink community space for posts and user interaction

## Technologies Used

- Flutter / Dart
- PHP
- MySQL
- Python
- MQTT
- TensorFlow Lite
- Google Colab
- Heltec LoRa V3
- DHT11 sensor
- Soil humidity sensor

## Project Goal

The goal of this project is to provide a smart agriculture solution that helps users monitor their environment, analyze collected data, receive useful recommendations, and diagnose plant diseases using image recognition.
